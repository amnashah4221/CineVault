CineVault is a modern movie discovery and management web application that allows users to search, explore, and save their favorite movies. It integrates an external movie API to provide detailed information about films including titles, categories, release dates, and ratings. Users can create their personalized movie collection and navigate seamlessly through the application.

## Features ‚ú®

- **Movie Search:** Search movies by title.
- **Category & Genre Filter:** Browse movies by categories, genres, or release year.
- **Favorites Management:** Save and view favorite movies in a personalized list.
- **Responsive Design:** Mobile-friendly layout with smooth navigation.
- **Dark Mode Toggle:** Switch between light and dark themes.
- **Detailed Movie View:** Access movie details including description, duration, rating, and release date,trailer, and cast.
